# Run

`Widgetbook` is supported in the following environments.

| Environment | Status | Comment                                                                                           |
| ----------- | ------ | ------------------------------------------------------------------------------------------------- |
| MacOS       | ✅     |
| Windows     | ✅     |
| Linux       | ✅     |
| Web         | ✅     | No hot reload, but hot restart (see [Issue 4](https://github.com/widgetbook/widgetbook/issues/4)) |
| Mobile      | ➖     | Will run but is not optimized. Let us know if you see a mobile use-case for `Widgetbook`.         |

See the [Desktop support for Flutter](https://docs.flutter.dev/desktop) page for
setup instructions.

Check if everything works by running `flutter run` with the newly created
Widgetbook:
