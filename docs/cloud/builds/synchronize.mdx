# Synchronize Builds

We understand that you manage your code with the help of a Git provider like GitHub, GitLab, Azure or others.
Widgetbook Cloud aims to seamingless integrate with the Git provider of your choice to make Widgetbook builds accessible to your team.
As we recommend to publish a Widgetbook build for each new Git commit, we aim to make Widgetbook builds accessible from commit.
Based on the Git provider, the implementation of the synchronization might differ and is described in the following sections.
