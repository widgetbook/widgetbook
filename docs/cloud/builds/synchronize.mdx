# Synchronize Builds

We understand that you manage most of your code work in a Git provider.
Widgetbook Cloud aims to seamingless integrate with your Git provider to make Widgetbook builds accessible for your team.
As we recommend to publish a Widgetbook build for each new Git commit, we aim to make Widgetbook builds accessible from commit.
Based on the Git provider, the implementation of the synchronization might differ and is described in the following sections.
