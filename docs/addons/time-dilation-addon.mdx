# Time Dilation Addon

A utility addon designed for modifying the scheduler's [`timeDilation`](https://api.flutter.dev/flutter/scheduler/timeDilation.html) property.

## Usage

```dart title=widgetbook/lib/widgetbook.config.dart
final config = Config(
  // ...
  addons: [
    TimeDilationAddon(), // [!code highlight]
  ],
);
```

## Mode

Each addon has a corresponding [Mode](/addons/modes) to lock its value in scenarios. For this addon, use `TimeDilationMode`:

```dart
final $Default = _Story(
  scenarios: [
    _Scenario(
      name: 'Slow Motion',
      modes: [
        TimeDilationMode(5.0), // [!code highlight]
      ]
    ),
  ],
)
```
