# Zoom Addon

The Zoom Addon is a Widgetbook utility designed for adding a zoom
use-case being previewed withing the Workbench.

## Setup

Here's how to configure it:

<Warning>
  In case `ZoomAddon` is used with other addons, then it must be defined before
  the [`AlignmentAddon`](/addons/alignment-addon) and after the
  [`DeviceFrameAddon`](/addons/device-frame-addon)
</Warning>

```dart
ZoomAddon(),
```
