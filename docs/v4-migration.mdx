# v4 Migration Guide

Since migrating to Widgetbook v4 involves a lot of breaking changes, it's advised to use an AI agent to automate the migration. An llm-friendly prompt for the migration is available [here](https://raw.githubusercontent.com/widgetbook/widgetbook/refs/heads/v4/docs/assets/guides/v4-migration/prompt.md).

If you prefer manual migration, you can bootstrap your new v4 project using our [CLI's init command](/cli/commands/init).
